Computers in the workforce

Computers are used in all different sorts of workplaces ranging from using computers in a library to using them in the hospital to store records.


Benefits

It can improve communication in the workplace as it allow people to talk with each other through the form of emails.Furthermore, less paper would be used helping out the environment as everything would be digital.


Risks

There is always a risk of data being hacked and stolen yet this can always be prevented with up to date security and preventions set in place. Current generation children are brought up with technology yet for older generation using computers everywhere may cause difficult as they would have to learn how to use them.
